cookie学习笔记

cookie简介

cookie是指web浏览器存储的少量数据，同时他是与具体的web页面或者站点相关的，cookie最早是设计为被服务端所用的，作为http协议的一种扩展实现它，cookie数据会自动在web浏览器和web服务器之前传输。

注意：操作cookie的API在浏览器上是完全兼容的，只不过设计的初期并没有考虑那么多，所以根本没有提供诸如查询、设置、删除的方法，所有的这些操作都要通过以特殊的字符串形式读写document的cookie属性来完成。每个cookie的有效期和作用域都可以通过cookie属性来分别指定。

cookie的具实现

cookie除了常用的名称和值之外，还有比较关键的可选属性来控制cookie的有效期和作用域。

  • 有效期

    cookie的默认有效期很短，它只能持续在web浏览器的会话期间，一旦用户关闭了浏览器，cookie保存的数据就丢失了。
并且cookie的有效期并不是局限在浏览器的单个窗口中，它的有效期和整个浏览器进程而非浏览器窗口的有效期。
如果要延长cookie的有效期，可以通过设置max-age属性来完成，但是必须明确的告诉浏览器cookie的有效期是多长，单位是秒。
一旦设置了有效期，浏览器就会将cookie数据存储在一个文件中，并且直到过了指定的有效期才会删除该文件。
expire属性与max-age同理，不通的是设置的步骤会多一步。

  • 作用域

    cookie的作用域是通过文档源和文档路径来确定的。该作用域通过cookie的path和domain属性来配置，
默认情况下，cookie和创建它的web页面有关，并对该web页面以及和该web页面同目录或者子目录的其他web页面可见。

比如：http://www.huzaixing.com/hzx/index.html页面创建了一个cookie，那么该cookie对http://www.huzaixing.com/hzx/
http://www.huzaixing.com/hzx/huzaixing/的所有web页面都是可见的。
但是对http://www.huzaixing.com/b.html、http://www.huzaixing.com/test/c.html等页面是不可见的。

  • path属性

    默认的cookie基本满足最常见的需求，有时候您可能希望改变这种配置，
例如很常见的你可能希望某一个页面创建的cookie在全网站都可以使用它，可以设置path属性，
例如：http://www.huzaixing.com/hzx/huzaixing/index.html页面创建了cookie，并且将path属性设置为/hzx，
那么对http://www.huzaixing.com/hzx/test/目录的web页面都是可见的。
如果把路径设置为"/",那么该cookie对任何http://www.huzaixing.com这台服务器上的页面都是可见的

注意：cookie的path属性不能被用做访问控制机制。如果一个web页面想要读取同一个站点其他页面的cookie，
只要简单的讲其他页面以隐藏iframe形式加载，随就可完全读取对应文档的cookie了。

  • domain属性

    cookie的作用域默认由文档源限制，很常见的就是大型网站会有很多子域名，想要在子域名之间互相共享cookie。
    比如：www.huzaixing.com域下的服务器想要读取v.huzaixing.com或者e.huzaixing.com域下设置的cookie值
    其实就是要在多个子域名之间共享cookie，将domian属性设置为.huzaixing.com即可。
    如果没有设置domain属性，那么域属性默认值是当前web服务器的主机名。
    注意：cookie的域只能设置为当前服务器的域。

  • secure属性

    secure属性表明cookie的值以何种形式通过网络传递。它是一个boolean值，cookie默认是以不安全的形式传递的，一旦是cookie被表示为
    安全的，那么只能当浏览器和服务器通过https或者其他的安全协议连接的时候才能传递它。

注意：在javascript中，cookie不会采用任何加密机制,因此他们是不安全的

cookie的增删查改

由于JavaScript原生的API并没有提供更多操作cookie的方式，所以cookie的变更都是以设置为一个字符串形式的值来进行的。

  • 新增cookie

    /**
*设置cookie
*parm name:cookie的名字
*parm value:cookie的值
*parm path:cookie的路径
*parm domain:cookie的域
*parm maxAge:cookie的有效时间        
/

function setCookie(name,value,path,domain,maxAge){
    var cookie = ""
    cookie += name + "=" encodeURIComponent(value);
    if(path){
        cookie += ';path="' + path + '"';
    }
    if(domain){
        cookie += ';domain="' + domain + '"';
    }
    if(domain){
        cookie += ';max-age="' + maxAge*60*60*24 + '"';
    }
     document.cookie = cookie;
}

  • 查询cookie

    /**
*获取指定cookie
*parm name:cookie的名字
*
*return string 指定cookie名称的值 
*    
/

function getCookie(name){
    var value = "";
    var cookies = document.cookie;
    if(cookies===""){
        return value;
    }
    var listCookie = cookies.split("; ");
    for(var i = 0 ; i < listCookie.length; i++){
        var thisCookie = listCookie[i];
        var cindex = thisCookie.indexOf("=");
        var cname = thisCookie.substring(0,cindex);
        var cvalue = thisCookie.substring(cindex+1);
        if(name == cname){
            value = cvalue;
            return encodeURIComponent(value);
            break;
        }
    }
    return value;
}

  • 删除cookie

    /**
*删除cookie
*parm name:cookie的名字
*通过设置指定cookie的值为空、有效时间为0来删除指定cookie;    
/
  function deleteCookie(name){
    var value = "";
    var cookies = document.cookie;
    if(cookies===""){
        return ;
    }
    var listCookie = cookies.split("; ");
    for(var i = 0 ; i < listCookie.length; i++){
        var thisCookie = listCookie[i];
        var cindex = thisCookie.indexOf("=");
        var cname = thisCookie.substring(0,cindex);
        var cvalue = thisCookie.substring(cindex+1);
        if(name == cname){
            document.cookie = name + '=""; max-age=0';
        }
    }
}

  • 清空cookie

    /**
*清空cookie
*通过设置指所有cookie的值为空、有效时间为0来删除指定cookie;    
/
  function clearCookie(){
    var value = "";
    var cookies = document.cookie;
    if(cookies===""){
        return ;
    }
    var listCookie = cookies.split("; ");
    for(var i = 0 ; i < listCookie.length; i++){
        var thisCookie = listCookie[i];
        var cindex = thisCookie.indexOf("=");
        var cname = thisCookie.substring(0,cindex);
        document.cookie = cname + '=""; max-age=0';
    }
}

注意： 标准浏览器不允许浏览器保存超过300个cookie，每个web服务器保存的cookie基本不能超过20个，并且每个cookie的数据不能
超过4KB（虽然现代浏览器运行cookie的总数超过300个）。cookie的值不允许分号、逗号以及空白符，因此在存储前一般采用
encodeURIComponent()对值进行编码，获取的时候使用decodeURIComponent函数进行解码

jquery cookie

鉴于原生的浏览器并没有提供诸如增删查改的具体实现，jquery.cookie提供了一套简介的API：

  • 获取cookie

    $.cookie(name) //指定的cookie名称

  • 设置cookie

    $.cookie(name,value) //指定的cookie名称以及值

  • 删除cookie 

    $.cookie(name,"") //指定的cookie名称以及值

  • 作用域以及过期时间、路径

    $.cookie(name,value,{path:"/",domain:"huzaixing.com",expires:"1"}) //过期时间单位是天

  • raw

    默认值：false。 默认情况下，读取和写入cookie的时候自动进行编码和解码（使用encodeURIComponent编码，decodeURIComponent解码）。
